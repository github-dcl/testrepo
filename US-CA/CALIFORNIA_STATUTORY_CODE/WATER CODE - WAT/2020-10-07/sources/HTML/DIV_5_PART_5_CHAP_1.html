Water Code - WAT

DIVISION 5. FLOOD CONTROL [8000 - 9651]

( Division 5 added by Stats. 1943, Ch. 369. )

PART 5. RECLAMATION BOARD BOND ACT [9250 - 9577]

( Part 5 added by Stats. 1943, Ch. 369. )

CHAPTER 1. Definitions [9250 - 9252]
( Chapter 1 added by Stats. 1943, Ch. 369. )

9250.

Unless the context otherwise requires the definitions in this chapter govern the construction of this part of this division of this code.

(Added by Stats. 1943, Ch. 369.)

9251.

Board means the reclamation board.

(Added by Stats. 1943, Ch. 369.)

9252.

Drainage district means the Sacramento and San Joaquin Drainage District.

(Added by Stats. 1943, Ch. 369.)
