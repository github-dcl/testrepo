Water Code - WAT

DIVISION 20. MUNICIPAL WATER DISTRICTS [71000 - 73001]

( Division 20 added by Stats. 1963, Ch. 156. )

PART 4. ELECTIONS [71450 - 71540]

( Part 4 added by Stats. 1963, Ch. 156. )

CHAPTER 3. Precincts and Polling Places [71473- 71473.]
( Chapter 3 added by Stats. 1963, Ch. 156. )

71473.

If any district election is consolidated with any state or county election, the voting precincts, polling places, and election officers for the district election shall be the same as those established for the state or county election.

(Added by Stats. 1963, Ch. 156.)
