Water Code - WAT

DIVISION 14. CALIFORNIA WATER STORAGE DISTRICT LAW [39000 - 48401]

( Division 14 added by Stats. 1951, Ch. 391. )

PART 8. BONDS [45100 - 45932]

( Part 8 added by Stats. 1951, Ch. 391. )

CHAPTER 2. Bond Election [45250 - 45278]

( Chapter 2 added by Stats. 1951, Ch. 391. )

ARTICLE 1. Notice [45250- 45250.]
( Article 1 added by Stats. 1951, Ch. 391. )

45250.

The notice of a special bond election shall state the aggregate face value of bonds proposed to be issued.

(Added by Stats. 1951, Ch. 391.)
