TITLE 2. GOVERNMENT OF THE STATE OF CALIFORNIA [8000 - 22980]

( Title 2 enacted by Stats. 1943, Ch. 134. )

DIVISION 4. FISCAL AFFAIRS [16100 - 17700]

( Division 4 added by Stats. 1945, Ch. 119. )

PART 1. FUNDS FOR SUBVENTIONS [16100 - 16202]

( Part 1 added by Stats. 1969, Ch. 1526. )

CHAPTER 2.5. Transfers to the State School Fund [16131- 16131.]
( Chapter 2.5 added by Stats. 1972, Ch. 1406. )

16131.

The funds appropriated by Section 16100 shall be used for any advances authorized by Section 17401.5 of the Education Code.

(Added by Stats. 1972, Ch. 1406.)
