Elections Code - ELEC

DIVISION 20. ELECTION CAMPAIGNS [20000 - 20502]

( Division 20 enacted by Stats. 1994, Ch. 920, Sec. 2. )

CHAPTER 5. Fair Campaign Practices [20400 - 20444]

( Chapter 5 enacted by Stats. 1994, Ch. 920, Sec. 2. )

ARTICLE 2. Definitions [20420- 20420.]
( Article 2 enacted by Stats. 1994, Ch. 920, Sec. 2. )

20420.

As used in this chapter, Code means the Code of Fair Campaign Practices.

(Enacted by Stats. 1994, Ch. 920, Sec. 2.)
